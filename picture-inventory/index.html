Picture Inventory

Share one photo or screenshot, run in-browser object detection, and review a numbered overlay with grouped inventory details.

45%

Setup: edit picture-inventory/Picture-Inventory.shortcut, replace YOUR_HOST (for this repo, use ejoliet.github.io), then import the shortcut on iOS and share a photo from Photos. Detection runs locally in the browser with a CDN-loaded COCO-SSD model; the confidence slider filters displayed detections after analysis.
